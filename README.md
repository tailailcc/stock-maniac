# stock-maniac
这是一个股票狂热分子写的软件；

这是一个有别于通达信、同花顺、大智慧等通用股软而为职业投资人所急需的软件。

我希望：

这是最好的股票分析辅助软件；

这是最好的python、javascript、vue教学软件；

这里蕴含着最具借鉴价值的股票交易思想。


股票很复杂，复杂到政治、经济、社会事件、行为心理、数量模型，无所不包，无所能包；

股票很简单，简单到只需做好两件事，交易正确的股票，正确地交易股票。

如果你对这个程序感兴趣，请将交流意见发送到lcclcq@sina.com，我将认真阅读与回复。

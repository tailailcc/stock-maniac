# stock-maniac
这是一个懂股票的程序员写的软件；

这是一个懂程序的证券投资者用于日常操盘、复盘、分析的软件；

这是一个弥补通达信、同花顺、大智慧等通用股软功能短板的软件；

这是一个可与通用股软联动的软件；

如果你对这个程序感兴趣，任何想法和意见都可以发送到835346610@qq.com，作者将认真阅读与回复。
